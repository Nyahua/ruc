# [开普勒定律](https://zh.wikipedia.org/wiki/%E5%BC%80%E6%99%AE%E5%8B%92%E5%AE%9A%E5%BE%8B)
> 开普勒第三定律，也稱為週期定律：各个行星绕太阳公转周期的平方及其椭圆轨道的半长轴的立方成正比。[1]

由这一定律不难导出：行星与太阳之间的引力与半径的平方成反比。这是艾萨克·牛顿的万有引力定律的一个重要基础。
