题目描述
【问题描述】
有n种砝码，它们的标准重量为w1, w2, …, wn克, 相应的数量分别为c1, c2, …, cn。砝码标准重量和数量可灵活组合，但已知它们称重的重物不会超过10000克。现有m个不同重量的物体需称重，请根据给定的砝码标准和数量确定每个重物被准确称量时，可采用的砝码组合方案数。
这里的组合方案指不同标准的砝码各有多少个，不区分具体是哪些砝码组合而成。比如，假设物体重10克，可用2个5克的砝码；5个2克的砝码；或者1个10克的砝码等多种组合方案。

【输入格式】
共4行，第1行两个数，分别为标准砝码的种类数n和物体数量m；
第2行，n个整数，分别为砝码的标准重量；
第3行，n个整数，与第2行不同标准砝码对应，依次为它们的数量；
第4行，m个整数，分别为m个待称重物体的重量，无重复。
所有数据均为空格分隔。

【输出格式】
根据物体称重的砝码组合方案数从多到少排序，逐行输出每个物体的重量及组合出该重量的砝码组合方案数；若组合数相同，按物体重量从小到大排。
输出时，每行2个整数，对应一个重物的重量和称重砝码组合方案， 数据用冒号，形如 5:7。表示5克重的物体，共有7种砝码组合方案可实现称重。
【数据规模】
砝码种类数 n  ≤  10；待称重物体数量 m  ≤  200；砝码组合数在千万量级。
