题目描述
完成排序函数mySort，可以对int数组的区间[Left, Right]进行从小到大排序
参数有3个，依次为：
待排序的数组ary
参与排序的数组下标左边界left
参与排序的数组下标右边界right 返回值类型为：void
例如： mySort(arary, 0, 10)表示对int数组array从arary[0] ~ array[10]的范围进行排序
本题为完成函数题，你不用处理输入输出。
